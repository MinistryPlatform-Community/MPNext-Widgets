Widget CodeAdd this to any page where you want this widget

<!-- Place in your header -->
<next-user-menu
  mp-base-url="__MP_BASE_URL__"
  post-logout-redirect-uri="[YourWebUrl]"

<!-- post-logout-redirect-uri defaults to current page if omitted -->

Universal SetupAdd ONCE to your site (e.g. header or layout template) — all widgets share this

<!-- Load MP Widgets (provides mpp-user-login) -->
<script id="MPWidgets" src="__MP_BASE_URL__/widgets/dist/MPWidgets.js"></script>

<!-- Load MPNext Embed SDK (auto-initializes) -->
<script type="module" src="https://your-host.com/embed-sdk/next-embed.js"></script>
