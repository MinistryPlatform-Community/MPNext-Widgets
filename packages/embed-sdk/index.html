MPNext Embed SDK

Widget Component Library

Prerequisites: Run pnpm dev in the project root to start the Next.js API backend on port 3000.

Widgets

Add to Calendar

<next-add-to-calendar>

Add-to-calendar buttons for events. Loads add-to-calendar-button library from CDN with Apple, Google, iCal, and Outlook options. Falls back to .ics download.

Full Calendar

<next-full-calendar>

Multi-view calendar with Month, Week, List, Cards (image grid), and Calendar (mini cal + cards) views. Supports event images, campus/ministry filter chips, density dots, admin detail modal, and configurable view locking.

User Menu

<next-user-menu>

Avatar dropdown with account modal. Shows the MP login widget when unauthenticated, then switches to an avatar menu after sign-in. Tabs: Profile, Family, Giving, Subscriptions, Invoices.

Adding a Widget to a Page

If the Universal Setup (below) is already on your site, adding a widget is just the HTML tag:

<!-- Just drop the tag on any page -- no extra scripts needed! -->
<next-full-calendar></next-full-calendar>

<!-- Multiple widgets work on the same page -->
<next-add-to-calendar event-id="12345"></next-add-to-calendar>
<next-user-menu></next-user-menu>

Universal Setup

Add this ONCE to your site (e.g. in your header or layout template). All widgets share this setup.

<!-- Load SDK -->
<script type="module" src="https://your-host.com/embed-sdk/next-embed.es.js"></script>

<!-- Initialize with token provider -->
<script type="module">
  import { init } from "https://your-host.com/embed-sdk/next-embed.es.js";

  init({
    tokenProvider: {
      get: async () => {
        const res = await fetch("/api/next-token");
        return (await res.json()).token;
      },
      refresh: async () => {
        const res = await fetch("/api/next-token?refresh=1");
        return (await res.json()).token;
      }
    }
  });
</script>
