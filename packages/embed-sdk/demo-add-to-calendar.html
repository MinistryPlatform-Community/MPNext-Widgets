Add to Calendar Widget Demo

Instructions:
  1. Make sure Next.js is running: pnpm dev in another terminal
  2. Enter a valid Event ID from Ministry Platform and click Reload Widget
  3. If the add-to-calendar-button CDN loads, you'll see a rich multi-calendar picker
  4. If the CDN is unavailable, a fallback .ics download button is shown
  5. Check the Event Log below for real-time updates

Widget Code — add this to any page

<!-- Use the widget -->
<next-add-to-calendar event-id="123"></next-add-to-calendar>

<!-- Listen to events -->
<script>
  const widget = document.querySelector("next-add-to-calendar");
  widget.addEventListener("calendarEventLoaded", (e) => {
    console.log("Event loaded:", e.detail);
  });
  widget.addEventListener("addToCalendarError", (e) => {
    console.error("Error:", e.detail);
  });
</script>

Universal Setup — add ONCE to your site (e.g. header or layout template)

<!-- Load SDK -->
<script type="module"
        src="https://cdn.northwoods.com/embed-sdk@1.0.0/next-embed.es.js"
        data-api-host="https://api.northwoods.com">
</script>

<!-- Initialize with token provider -->
<script type="module">
  import { init } from "https://cdn.northwoods.com/embed-sdk@1.0.0/next-embed.es.js";

  init({
    tokenProvider: {
      get: async () => {
        const res = await fetch("/api/next-token", {
          method: "POST",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify({ wid: "add-to-calendar" })
        });
        return (await res.json()).token;
      }
    }
  });
</script>
